## 阶段 1：通信链路验证 ⏳

> 工具：VOFA+（串口直发 Modbus RTU 帧）
> 接口：X2 RS232（Pr5.37=1, 站号=1, 9600bps, 8E1）

### Step 1.1 — 回环测试（确认链路物理层通）

发送帧（VOFA+ 十六进制发送）：
```
01 08 00 00 12 34 ED 7C
```
预期：驱动器**原样回发**同样 8 字节。
- ✅ 收到回发 → 物理链路 OK，进入 Step 1.2
- ❌ 无回应 → 检查 TXD/RXD 交叉接线、COM 口号、波特率 8E1

### Step 1.2 — 读取状态字（确认 Modbus 从站响应）

发送帧：
```
01 03 40 00 00 01 91 CA
```
预期：回包格式 `01 03 02 XX XX CRC_L CRC_H`，其中 XX XX 是 Statusword。
- Statusword bit15=0 → 伺服未使能（正常）
- Statusword bit15=1 → 伺服已使能

### Step 1.3 — 伺服使能 ON（确认控制链路通）

> **前提：** 电机已接线、驱动器已上电、无报警（PANATERM 确认）

发送帧：
```
01 05 00 60 FF 00 8C 24
```
预期：电机抱闸松开，可手动盘轴感觉阻力变化。
- 读 Statusword 验证：bit15 应变为 1

发送伺服 OFF：
```
01 05 00 60 00 00 CD D4
```

### Step 1.4 — 写入速度档位 V0 + 加减速档位 A0/D0

写 V0 = 1000 r/min（4600h，功能码 0x06）：
```
01 06 46 00 03 E8 XX XX
```
> XX XX 为 CRC16，需用工具计算后填入

写 A0 = 200 ms/(3000r/min)（4610h）：
```
01 06 46 10 00 C8 XX XX
```

写 D0 = 200 ms/(3000r/min)（4620h）：
```
01 06 46 20 00 C8 XX XX
```

### Step 1.5 — 写 Block 0（相对定位 +10000 pulse）并启动

Block 0 数据（详见 block_指令格式.md 第八节示例）：
```
Block command = 0x01000000  （命令01h，V0/A0/D0，迁移条件=结束）
Block data    = 0x00002710  （+10000 pulse）
```

写 Block 0（4800h~4803h，功能码 0x10）：
```
01 10 48 00 00 04 08  01 00 00 00 00 00 27 10  XX XX
```

指定 Block 0（写 4414h = 0000h）：
```
01 06 44 14 00 00 XX XX
```

触发 STB（写线圈 0120h = FF00h）：
```
01 05 01 20 FF 00 XX XX
```

预期：电机转动一段距离后停止（BUSY 变 0）。
