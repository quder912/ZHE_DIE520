# VOFA+ 控制 MSMF022L1U2M + MADLT15SF 正反转10圈 操作指引

## 设备信息

| 项目 | 值 |
|------|-----|
| 电机 | MSMF022L1U2M（200W，额定3000r/min） |
| 驱动器 | MADLT15SF（A6SF多功能型） |
| 编码器 | 23bit绝对式 = **8,388,608 pulse/圈** |
| 通信 | USB转RS232 → X2口，Modbus RTU |

---

## 第一步：驱动器参数（前面板设定，设定后重启电源）

| 参数 | 值 | 含义 |
|------|----|------|
| **Pr0.01** | **0** | 位置控制模式 |
| **Pr5.37** | **1** | Modbus-RTU（RS232） |
| **Pr5.29** | **6** | RS232波特率=115200bps |
| **Pr5.38** | **4** | None奇偶 / 1bit停止位 |
| **Pr5.31** | **1** | 从站地址=1 |
| **Pr6.28** | **1** | Modbus通信启动Block动作 |

> 前面板操作：MODE键→参数模式→↑↓选号→SET进入→改值→SET确认→全部改完**断电重启**

---

## 第二步：接线

```
USB-RS232        X2口
  TXD ──────────→ 4 RXD    (交叉)
  RXD ←────────── 3 TXD    (交叉)
  GND ─────────── 1 GND
```

> 在设备管理器确认COM口号

---

## 第三步：VOFA+ 串口设置

| 项目 | 值 |
|------|-----|
| 协议 | Modbus RTU |
| COM口 | 你的COM号 |
| 波特率 | 115200 |
| 数据位 | 8 |
| 奇偶 | None |
| 停止位 | 1 |
| 从站地址 | 1 |

---

## 第四步：Block编程方案

```
Block 0: 相对定位 左旋10圈（-83886080 pulse）→ 完成后迁移
Block 1: 相对定位 右转10圈（+83886080 pulse）→ 完成后迁移
Block 2: 跳转到Block 0 → 无限循环
```

停止方式：发送S-STOP命令。观察到约100次后手动停止。

---

## 第五步：按顺序发送以下Modbus帧

### 1. 获取执行权
```
01 06 43 00 00 55 5C 71
```
（写寄存器4300h=0055h）

### 2. 伺服ON
```
01 05 00 60 FF 00 8C 24
```
（写线圈0060h=FF00h）

### 3. 速度[0]=500r/min
```
01 06 46 00 01 F4 9C 95
```
（写寄存器4600h=01F4h）

### 4. 加速时间[0]=200ms
```
01 06 46 10 00 C8 9C D1
```
（写寄存器4610h=00C8h）

### 5. 减速时间[0]=200ms
```
01 06 46 20 00 C8 9C DE
```
（写寄存器4620h=00C8h）

### 6. 写Block 0（左旋10圈）
```
01 10 48 00 00 04 08 01 00 03 00 FB 00 00 00 A7 29
```
（写4800h~4803h，command=0x01000300，data=0xFB000000即-83886080）

### 7. 写Block 1（右转10圈）
```
01 10 48 04 00 04 08 01 00 03 00 05 00 00 00 67 0E
```
（写4804h~4807h，command=0x01000300，data=0x05000000即+83886080）

### 8. 写Block 2（跳转Block 0）
```
01 10 48 08 00 04 08 09 00 02 00 00 00 00 00 77 A5
```
（写4808h~480Bh，command=0x09000200，data=0x00000000）

### 9. 指定Block编号0
```
01 06 44 14 00 00 DD 3E
```
（写寄存器4414h=0000h）

### 10. STB=ON 启动！
```
01 05 01 20 FF 00 8C 0C
```
（写线圈0120h=FF00h，电机开始正反转循环）

---

## 第六步：停止（观察到约100次后发送）

### 11. S-STOP 减速停止
```
01 05 01 24 FF 00 CD CD
```
（写线圈0124h=FF00h）

### 12. 伺服OFF
```
01 05 00 60 00 00 CD D4
```
（写线圈0060h=0000h）

### 13. 释放执行权
```
01 06 43 00 00 AA 1C 31
```
（写寄存器4300h=00AAh）

---

## 监视器（随时可读取）

| 用途 | 帧 |
|------|-----|
| 读实际速度(4D06h) | `01 03 4D 06 00 02 33 66` |
| 读错误代码(4001h) | `01 03 40 01 00 01 C0 0A` |
| 读BUSY状态(0140h) | `01 01 01 40 00 01 FD E2` |

---

## 报警清除（如遇报警）

```
01 06 41 02 72 74 XX XX
```
（写寄存器4102h=7274h清除报警，CRC需重新计算）

---

## 时间估算

- 500r/min → 约1.2秒/10圈（含加减速约1.6秒）
- 正反1次约3.2秒
- **100次约5.3分钟**

---

## 注意事项

1. **先设参数再重启**：Pr5.37、Pr5.29、Pr6.28都是电源重启属性
2. **TXD接RXD交叉**：不要接反
3. **先获取执行权**：必须先发Step 1
4. **首次测试建议低速**：500r/min安全，确认正常后可提高到1000/2000
5. **紧急停机**：发H-STOP(`01 05 01 23 FF 00`) 或 伺服OFF
6. **STB复位**：建议设Pr5.42 bit2=1让STB自动OFF，否则需手动写0120h=0000h
7. **参数修改需停Block**：Block动作中改参数不保证生效
