寄存器映射
==========

本文件记录松下 MINAS A6 系列驱动器的 Modbus 寄存器地址映射。

1. 线圈地址（Modbus 0x01/0x05 功能码）

   地址     名称       说明
   ─────── ────────── ─────────────────────────
   0x0060  Servo ON   伺服使能（ON=1, OFF=0）
   0x0120  STB        Block 动作启动触发（Pr5.42=4 时自动 OFF）

2. 保持寄存器地址（Modbus 0x03/0x06/0x10 功能码）

   保持寄存器的寻址范围：
   - 驱动器参数：Pr0.xx ~ Pr7.xx
   - 控制命令寄存器
   - 状态监控寄存器
   - Block 动作相关寄存器

   各 Pr 参数对应的 Modbus 寄存器地址如下：

   参数编号   寄存器地址   说明
   ──────── ──────────── ─────────────────────
   Pr0.00    0x0000      驱动器参数区域
   Pr0.01    0x0001
   Pr0.02    0x0002
   ...
   Pr0.13    0x000D      转矩限制（Pr0.013, 对应 PDF 中转矩限制设定）
   ...
   Pr5.00    0x0003      站号（Modbus 从站地址）
   Pr5.01    0x0004      通信速度
   Pr5.02    0x0005      通信协议选择
   Pr5.03    0x0006      数据格式
   Pr5.42    0x002A      STB 自动 OFF 设置
   ...
   Pr6.28    0x006C      通信控制选择
   Pr6.88    0x0158      绝对式多圈数据上限值

   注意：PDF 中的寄存器地址表页（第 39~55 页）信息以图片/表格形式存在，
   提取受限。上述地址基于 PDF 文档片段整理，使用前建议用 Panaterm 验证。

3. PDF 中提到的关键寄存器区域（寄存器一览页面 41-49）

   - 控制命令寄存器区域（用于发送控制指令）
   - 参数读写区域（用于读写 Pr 参数）
   - Block 动作数据区域（用于编辑 Block 动作指令）
   - 状态监控区域（用于读取运行状态）
   - 告警记录区域

   寄存器一览表中标记为 "-" 的部分表示保留或未使用的地址。

4. Block 动作相关寄存器

   PDF 文档提到通过 4414h 寄存器写入 Block No：
   - 4414h（0x4414）：写入要启动的 Block 编号
     当通过 Modbus 启动 Block 动作时，向此地址写入 Block 编号

5. 状态监控寄存器（bit 信息）

   PDF 第 50-55 页提供了状态寄存器的 bit 位定义，包括：
   - 32 位状态寄存器（地址 0x0260-0x0261 等）
   - 各 bit 含义：伺服使能状态、告警状态、定位完成、转矩限制中、速度到达等

6. 计算公式

   实际值 = 寄存器值 × 换算系数

   位置反馈：脉冲数（取决于电子齿轮比 Pr0.008/Pr0.009）
   速度反馈：单位 r/min（具体换算系数需查主手册）
   转矩反馈：单位 %（额定转矩百分比）
